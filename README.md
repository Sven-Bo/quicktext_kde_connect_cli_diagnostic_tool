# KDE Connect Diagnostic Tool

Diagnostic tool for troubleshooting KDE Connect CLI issues with QuickText.

## What It Does

This batch script checks if KDE Connect CLI is properly configured for use with QuickText Excel tool.

It tests:
- Command execution in CMD and PowerShell
- File existence (Microsoft Store and Desktop versions)
- PATH environment variable configuration
- System information

## How to Use

1. Download [kde_connect_diagnostic.bat](cci:7://file:///c:/Users/codin/Desktop/New%20folder/kde_connect_diagnostic.bat:0:0-0:0)
2. Double-click to run
3. Share the log file with QuickText support

The tool creates a log file at `%TEMP%\quicktext_kde_diagnostic.log` and automatically opens the folder when finished.

## Requirements

- Windows 10/11
- No admin rights needed
- Safe to run (read-only)
